# Instrucciones para iniciar el servidor de desarrollo

## Paso 1: Abre una terminal PowerShell

## Paso 2: Navega al directorio frontend
```powershell
cd C:\Trabajo\BalconDemo\frontend
```

## Paso 3: Ejecuta el servidor de Vite
```powershell
npm run dev
```

## Resultado esperado:
Deberías ver algo como:
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network:  use --host to expose

  ready in xxx ms.
```

## ⚠️ IMPORTANTE:
- **NO cierres esta terminal** mientras trabajas
- El servidor debe mantenerse corriendo
- Deja esta terminal abierta

## Si ves errores:
1. Asegúrate de estar en `C:\Trabajo\BalconDemo\frontend`
2. Verifica que las dependencias estén instaladas: `npm install`
3. Verifica que no haya otro proceso usando el puerto 5173
